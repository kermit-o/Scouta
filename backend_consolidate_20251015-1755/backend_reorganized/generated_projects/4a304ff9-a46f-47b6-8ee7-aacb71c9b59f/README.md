# Portal de Noticias Generado

## Descripción
Proyecto generado automáticamente por Forge SaaS.

## Características
- Frontend: React con TypeScript
- Backend: Node.js con Express
- Base de datos: MongoDB
- Autenticación: JWT

## Instalación
\`\`\`bash
npm install
npm run dev
\`\`\`

## Estructura
- `/src` - Código fuente
- `/public` - Archivos estáticos
- `/api` - Endpoints backend
